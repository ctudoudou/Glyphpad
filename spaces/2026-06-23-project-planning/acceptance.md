# Acceptance Criteria

This planning iteration is complete when:

- `AGENTS.md` defines the durable project workflow.
- The first planning space exists under `spaces/`.
- Branch management is documented.
- Requirement tracking through spaces is documented.
- Commit discipline is documented.
- SQLite is explicitly recorded as the local storage baseline.
- The initial product direction distinguishes the Launchpad surface from the control panel.
- No application scaffold or implementation code is introduced.

Manual verification:

- Confirm the repository contains only planning artifacts after this iteration.
- Confirm future work has an obvious place to record scope, decisions, TODOs, and acceptance criteria.
