# Decisions

## Document Current Behavior Only

The README describes automatic classification as a future workflow with API settings already present. It does not claim that LLM classification is fully implemented.

## Single Verified Screenshot

Only the verified launcher screenshot is included. An attempted settings screenshot did not show the settings window, so it was removed rather than documenting a misleading image.

## Separate Chinese README

Chinese documentation lives in `README.zh-CN.md` and links back to the English README. The English README links to it at the top.

## Size Claim

The README describes the local app bundle as about 3 MB because the current generated `dist/Glyphpad.app` measures 2.8 MB. This keeps the statement useful without implying a fixed release size.
